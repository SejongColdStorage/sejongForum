## 기능리스트

- 사용자(User)는 회원가입이 가능하다.
- 게시판(Forum)은 용도에 따라 생성이 가능하다.
- 글(Post)을 게시판에 작성 및, 수정이 가능하다.
- 댓글(Comment)을 글마다 작성할 수 있다.
